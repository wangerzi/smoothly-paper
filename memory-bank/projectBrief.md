# 项目简介 - 论文流畅读 (Smoothly Paper)

## 项目概述

**论文流畅读（Smoothly Paper）** 是一个 AI 驱动的英文学术论文阅读辅助工具，专为需要阅读英文论文但英语能力有限的中文学术研究者设计。

## 核心价值

帮助用户通过 AI 智能分析，逐步提升专业英语阅读能力，降低学术论文的阅读门槛。

## 目标用户

- 研究生新手（22-26岁，CET-4/6 水平）
- 科研工作者（28-40岁，需要快速筛选文献）
- 跨领域学习者（25-35岁，面临新领域术语壁垒）

## 核心功能（MVP v1.0）

### 1. PDF 上传与解析
- 用户可上传学术论文 PDF（≤20MB）
- 自动提取文本内容
- 支持拖拽和点击上传

### 2. 个性化学习设置
- 三档词汇水平选择：初级（3K词汇）/ 中级（5K词汇）/ 高级（8K+词汇）
- 根据水平智能调整难词标注密度

### 3. AI 智能分析
- **论文总结**：生成 200-300 字的结构化中文摘要
- **智能分段**：识别论文结构，拆分为易读小段（150-300词/段）
- **多维度辅助**：
  - 🔤 专业术语解释
  - 📚 难词标注和释义
  - 🔍 复杂句法分析
  - 🌏 准确的段落翻译

### 4. 沉浸式阅读界面
- 三栏布局（目录 + 原文 + 辅助面板）
- 生词点击查看详细解释
- 实时阅读进度追踪
- 护眼模式和字号调节

## 技术特点

### 本地优先架构
- **完全本地运行**：除 AI API 外，所有功能在本地执行
- **数据隐私**：论文数据存储在本地，不上传云端
- **零配置启动**：开箱即用，无需复杂设置

### 现代化技术栈
- **前端**：Next.js 14 + React + TypeScript + Tailwind CSS
- **UI 库**：shadcn/ui（炫酷的赛博朋克风格）
- **后端**：Next.js API Routes（本地运行）
- **数据库**：SQLite（本地文件数据库）
- **PDF 处理**：pdf-parse（文本提取）+ pdf.js（渲染）
- **AI 服务**：OpenAI GPT-4 API（唯一外部依赖）

### 炫酷视觉设计
- 赛博朋克风格：深空色调 + 霓虹点缀
- 毛玻璃效果和渐变光晕
- 粒子背景动画
- 流畅的微交互动效
